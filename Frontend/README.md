# Frontend — Ruvia Cosmetics (Next.js App Router)

This folder contains the Next.js storefront + admin UI.

For the **full system architecture, backend/API, tradeoffs, and production readiness**, see the top-level README:
- [../README.md](../README.md)

## Quick start
```bash
npm install
npm run dev
```

## Useful scripts
- `npm run dev` — dev server (Turbopack)
- `npm run dev:webpack` — dev server (Webpack fallback)
- `npm run build` — production build
- `npm run analyze` — bundle analysis build (`ANALYZE=true`)
