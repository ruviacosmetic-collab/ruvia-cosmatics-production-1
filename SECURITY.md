# Security Policy

## Supported Versions

Only the latest stable version of the Ruvia Cosmetics digital platform is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |

## Reporting a Vulnerability

As a medical-led brand, we take digital security and user data privacy with the utmost seriousness. If you discover a security vulnerability, please report it immediately via the following process:

1. **Email**: Security reports should be sent to security@ruviacosmetics.com.
2. **Details**: Please include a detailed description of the vulnerability and steps to reproduce.
3. **Response**: You will receive an acknowledgment within 24 hours.

Please do not disclose the vulnerability publicly until it has been resolved.
